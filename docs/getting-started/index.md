# Getting Started

Bring up a fresh macOS box on this dotfiles setup, move existing dotfiles aside before stowing,
and tweak the bits that vary per machine.

| Page                      | Purpose                                                                |
| ------------------------- | ---------------------------------------------------------------------- |
| [Install](install.md)     | Run the staged installer. Each stage is idempotent and can be re-run.  |
| [Backup](backup.md)       | Move existing dotfiles aside so `stow` won't refuse to symlink.        |
| [Customize](customize.md) | Env vars (`REPOS_DIR`, etc.) and overlay paths for per-machine tweaks. |
